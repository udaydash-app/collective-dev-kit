# data/ — Local Database & Edge Functions Bundle

This folder is shipped inside the Windows installer as `resources/data/`.

Contents (generated by `scripts/build-local-db.mjs`):
- `schema.sql` — Postgres schema dumped from cloud, RLS/grants stripped, FKs to `auth.users` removed.
- `seed.sql`   — Data dump from cloud at build time (products, contacts, accounts, pos_users, etc.).
- `pglite/`    — Pre-built PGlite (Postgres-in-WASM) data directory. Copied to the user's
                 `userData/data/pglite` on first launch so the app starts with a ready DB.
- `functions/` — (Phase B) Ported edge functions executed by Electron main process when offline.

## Rebuild

```bash
DATABASE_URL='postgresql://postgres:PASSWORD@db.wvdrsofehwiopbkzrqit.supabase.co:5432/postgres' \
  node scripts/build-local-db.mjs
```

Then rebuild the Electron installer; `data/` is included via `build.extraResources`.
